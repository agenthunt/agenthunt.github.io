Waterline to GraphQL
Waterline is a great ORM that abstracts different databases using an uniform API. One of the problems with waterline and in general REST services is the need to create multiple endpoints for different data requirements. Querying data often ends up being multiple round trip requests. GraphQL solves this problem by making the client specify data requirements and retrieve the exact data in a single request. Waterline to GraphQL adapter is an attempt to marry the best of both worlds. With this adapter, you can use all your existing backends supported by waterline and the client can make graphql requests.
